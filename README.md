## 🚀 Quick Start

1. **Upload Theme** -- Shopify Admin > Online Store > Themes > Upload `prm-herbovilla-theme.zip`
2. **Publish Theme** -- Preview and publish
3. **Upload Logo** -- Theme Customizer > Settings > Upload `Prm Logo.jpg`
4. **Import Collections** -- Products > Collections > Import `shopify-collections-import.csv`
5. **Create Pages** -- Follow [THEME-UPLOAD-WALKTHROUGH.md](THEME-UPLOAD-WALKTHROUGH.md)
6. **Set Up Collections** -- Follow [COLLECTION-SETUP-GUIDE.md](COLLECTION-SETUP-GUIDE.md)
7. **Add Products** -- Upload with images from `All product 3d photo/`
