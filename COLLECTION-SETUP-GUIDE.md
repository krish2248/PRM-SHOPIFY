## Step 1: Access Collections

1. Log in to `prm-herbovilla.myshopify.com/admin`
2. Click **Products** in the left sidebar
3. Click **Collections**
4. Click **Create collection** button

## Step 3: Add Products to Collections

### Method A: From Collection Page
1. Open a collection
2. Click **Add products**
3. Search and select relevant products
4. Click **Save**

### Method B: From Product Page
1. Open a product
2. Scroll to **Collections** section
3. Select collections this product belongs to
4. Click **Save**

### Method C: Bulk Edit
1. Go to **Products → All products**
2. Select multiple products using checkboxes
3. Click **Bulk edit**
4. Add collection tags to multiple products at once

---

## Step 4: Add Collection Images

For each collection:
1. Open the collection
2. Click **Add image** under Collection image
3. Upload a high-quality image (recommended: 1200x600px)
4. Use lifestyle images or product group photos

**Image Suggestions**:
- Immunity: Fresh herbs, tulsi leaves
- Digestive: Triphala ingredients, healthy gut imagery
- Hair & Skin: Herbal oils, natural ingredients
- Joint Care: Active lifestyle, herbal ingredients
- Women's Health: Soft florals, shatavari plant
- General Wellness: Holistic health imagery

## Step 5: Set Collection Order in Navigation

1. Go to **Online Store → Navigation**
2. Click on **Main menu**
3. Add menu item:
   - Name: Shop
   - Link: Select collection → All Products
4. Add nested items for each collection category
5. **Save menu**

### Suggested Menu Structure:
```
Main Menu
├── Home
├── Shop
│   ├── All Products
│   ├── Best Sellers
```

## Step 6: Feature Collections on Homepage

After collections are created:
1. Go to **Online Store → Themes → Customize**
2. Find **Shop by Goal** section
3. Select your collections to display
4. Arrange in preferred order
5. **Save**

---

## Pro Tips

1. **Use Automated Collections**: For dynamic collections like "Under ₹500" or "New Arrivals", use automated rules based on price, tags, or date added.

2. **Add Tags to Products**: Tag products with relevant keywords (immunity, digestive, etc.) for easier filtering.

3. **Cross-list Products**: A product can belong to multiple collections. For example, Ashwagandha can be in both "Immunity" and "Men's Health".

4. **Regular Updates**: Keep "Best Sellers" and "New Arrivals" collections updated monthly.

5. **Collection SEO**: Fill in SEO title and description for each collection for better search visibility.
